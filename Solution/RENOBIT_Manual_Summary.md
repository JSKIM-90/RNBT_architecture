## 1. RENOBIT 소개

### 개요
**"2D, 3D 컴포넌트 기반의 저작도구를 포함하는 웹 표준 통합관제솔루션"**

RENOBIT은 웹 표준 기술을 활용한 하이브리드 저코드 플랫폼으로, 2D 및 3D 컴포넌트 기반의 웹 애플리케이션 개발을 지원합니다.

### 주요 특징

#### 1. 웹 표준 기반 저코드 하이브리드 플랫폼
- **HTML, CSS, JavaScript** 활용으로 콘텐츠와 디자인의 효과적 분리 및 통합
- **UI 기반 레이아웃 작업**으로 개발 생산성 극대화
- **간소화된 작업**과 **코드 유연성** 결합

**장점**:
- 비개발자도 시각적으로 페이지 구성 가능
- 필요시 코드 직접 작성으로 세밀한 제어 가능
- 개발 시간 단축 및 유지보수 용이

#### 2. 외부 디자인 플랫폼 통합
- 디자인 도구에서 **내보내기 코드를 효율적으로 통합**
- **디자이너와 개발자의 역할 분담**으로 협업 효율성 향상

**워크플로우**:
```
디자인 도구 (Figma, Sketch 등)
→ 코드 내보내기
→ RENOBIT 통합
→ 개발자 커스터마이징
```

#### 3. 3D 속성 UI 지원
- **위치, 회전, 크기, 재질** 등을 UI로 직관적 조작
- **코딩 작업 감소**로 생산성 향상
- Three.js 기반 3D 렌더링

**지원 기능**:
- X/Y/Z 축 위치 조정
- 회전 각도 설정
- 스케일 조정
- 재질 및 텍스처 적용

#### 4. 박스모델 스타일링 UI
- **margin, padding, border** 등을 UI로 직관적 설정
- 시각적 피드백으로 빠른 레이아웃 작업

### 라이선스 유형

#### Named User
- **특정 사용자 지정** 방식
- **최초 30명**부터 시작, **10명 단위로 추가** 가능
- 개인별 계정 할당으로 사용 이력 추적 용이

#### Concurrent User
- **동시 접속 제한** 방식
- **최초 10명**부터 시작, **5명 단위로 추가** 가능
- 동시 사용자 수 기반 비용 최적화

#### Trial
- **30일 제한**, **사용자 제한 없음**
- 평가 및 테스트 용도

### 라이선스 처리 정책

**위반 상황**:
- 기간 초과
- IP 주소 변경
- 중복 접속
- 동시 접속 초과

**처리 방식**:
- 로그 작성 및 경고 메시지 표출
- 기간 만료 시 **라이선스 키 재등록 필요**

## 2. 에디터 매뉴얼

### 개요
RENOBIT은 웹 기반 페이지 편집 도구로, 로그인 후 에디터 또는 뷰어 화면에 접근할 수 있습니다.

### 로그인 및 권한
- **아이디와 비밀번호** 입력으로 시스템 접속
- **편집 권한이 있는 사용자만** 에디터 접근 가능
- 권한 없는 사용자는 뷰어 모드로 제한

### UI 구성 요소

#### 상단 메뉴
- **신규 페이지 생성**, **저장** 기능
- **페이지/리소스 가져오기 및 내보내기**
- **통합 가져오기/내보내기** 기능
- **로그아웃** 옵션

#### 편집 및 레이어 툴바
- **컴포넌트 복사/오려두기/붙여넣기/삭제**
- **레이어 표시/숨김 설정**
- **줌 인/아웃** 기능

#### 메뉴 토글바
- **상단, 좌측, 우측 메뉴** 펼침/접음 기능
- 작업 공간 최적화

### 주요 패널

#### 페이지 패널
- **트리 구조**로 페이지 관리
- **폴더 생성** 가능
- 페이지 계층 구조 시각화

**활용 예시**:
```
프로젝트
├── 메인 페이지
├── 서브 페이지
│   ├── 페이지 A
│   └── 페이지 B
└── 관리 페이지
```

#### 컴포넌트 패널
- **2D/3D 컴포넌트** 목록 제공
- **드래그앤드롭**으로 캔버스에 배치
- 컴포넌트 분류별 정리

#### 인스턴스 패널
- 배치된 **컴포넌트 선택**
- **검색 기능**
- **가시성 제어**
- 인스턴스별 속성 확인

#### 그룹 패널
- **2D 레이어의 그룹 컴포넌트** 표시 및 관리
- 계층 구조 관리

#### 매니저 패널
**데이터셋 관리자**:
- API 엔드포인트 설정
- 데이터 소스 관리

**리소스 관리자**:
- 이미지, 폰트, 3D 모델 등 관리
- 파일 업로드 및 참조

**다국어 관리자**:
- 다국어 키-값 쌍 관리
- 언어별 번역 관리

#### CodeBox 패널
- **HTML, CSS, JS 통합 작성** 인터페이스
- 코드 자동완성 및 구문 강조
- 실시간 미리보기

#### GUI 패널
- **3D 작업용 속성 및 정보 관리**
- 3D 객체 변환, 재질, 조명 설정

### 페이지 속성

#### 기본 설정
- **문서 ID**: 고유 식별자
- **이름**: 페이지 이름
- **Z-인덱스**: 레이어 순서

#### 마스터 페이지 연동
- **참조/마스터 페이지 관리**
- 공통 레이아웃 재사용

**활용 사례**:
- 헤더/푸터 공통화
- 네비게이션 일관성 유지

#### 레이아웃 설정
**크기 및 단위**:
- **너비/높이** 설정
- **단위 선택**: PX, %, EM, REM, VW, VH 등

**스타일링**:
- **경계선 굴곡도** (border-radius)
- **여백** (margin)
- **패딩** (padding)
- **Display**: block, inline, flex, grid 등
- **Overflow**: visible, hidden, scroll, auto

#### Flexbox 설정
**활성화 조건**: `display: flex` 시

**옵션**:
- **방향** (flex-direction): row, column 등
- **줄바꿈** (flex-wrap): nowrap, wrap
- **정렬** (justify-content, align-items)

**사용 예시**:
```css
display: flex;
flex-direction: row;
justify-content: space-between;
align-items: center;
```

#### Grid 설정
**활성화 조건**: `display: grid` 시

**옵션**:
- **열/행 템플릿** (grid-template-columns/rows)
- **자동 배치**
- **간격** (gap)

**사용 예시**:
```css
display: grid;
grid-template-columns: repeat(3, 1fr);
gap: 20px;
```

#### 3D 속성
- **X/Y/Z 축 위치**
- **그리드 간격 제어**
- 3D 씬 설정

### 컴포넌트 속성
- **기본 인스턴스명** 표시
- **각 컴포넌트별 맞춤 속성** 설정 가능
- 동적 속성 바인딩

- 반응형 이미지

- 정렬 가능 테이블

- 스트라이프, 호버 효과

## 4. Code Box 매뉴얼

### 개요
**Code Box**는 HTML, CSS, JavaScript 영역을 분리하여 체계적으로 코드를 관리할 수 있는 환경을 제공합니다.

**주요 기능**:
- **코드 자동 완성**
- **구문 강조**
- **실시간 미리보기**
- 개발 속도 향상

### Header 기능

#### Instance Name & Id
- 선택한 **페이지/컴포넌트 이름** 표시
- **클릭 시 아이디가 클립보드에 복사**됨
- 빠른 참조 및 스크립팅

#### Instance Path
- 컴포넌트의 **배치 경로 표시**
- **클릭 시 해당 위치로 이동**
- 계층 구조 탐색 용이

#### Format Code
- **에디터 규칙에 맞게 코드 정렬**
- 들여쓰기 자동 조정
- 코드 가독성 향상

#### Apply
- **올바른 코드를 해당 인스턴스에 즉시 적용**
- 실시간 반영
- 에러 발생 시 경고 표시

### Side 기능

#### Instance List
- 현재 페이지의 **인스턴스를 2D/3D 레이어별로 리스트** 형식 표시
- 빠른 인스턴스 전환

**표시 형식**:
```
2D Layer
├── Header
├── ContentArea
└── Footer

3D Layer
├── SceneObject1
└── SceneObject2
```

#### Group Item List
- 선택한 컴포넌트에 **포함된 인스턴스 목록** 표시
- 그룹 내 요소 관리

#### Instance Options
**리소스 URL 가져오기**:
- 이미지, 폰트, 3D 모델 경로 자동 삽입

**폰트 리스트**:
- 사용 가능한 폰트 목록
- 폰트 적용 코드 스니펫

**클래스명 관리**:
- 클래스 추가/수정/삭제
- CSS 클래스 참조

#### Default Code Snippet
**제공 스니펫**:
- **Animation**: 애니메이션 코드
- **Page**: 페이지 스크립트
- **Instance**: 인스턴스 접근
- **Data Service**: 데이터 fetch
- **Locale**: 다국어 처리
- **Resource**: 리소스 로드

**예시 (Data Service)**:
```javascript
// 데이터 fetch 스니펫
const { fetchData } = WKit;

fetchData({
  datasetName: 'api',
  param: { endpoint: '/users' }
}).then(data => {
  console.log(data);
});
```

#### Sample Code
- 선택된 **컴포넌트 관련 코드 스니펫** 제공
- 컴포넌트별 맞춤 예제

#### SVG Style
**Color Theme**:
- **Mask**: SVG 마스크 적용
- **Hex**: 색상 Hex 코드
- **RGB**: RGB 색상 값
- **Color Matrix**: 색상 매트릭스 변환

**Animation**:
- SVG 애니메이션 효과

### Box 영역

#### 레이아웃 구성
- **헤더 드래그**로 영역 이동 및 배치 가능
- 자유로운 레이아웃 커스터마이징

**작성 가능 영역**:
- 컴포넌트 특성에 따라 다름
- **모든 컴포넌트**: HTML/CSS/JS 모두 작성
- **일부 컴포넌트**: CSS/JS만 작성
- **스크립트 컴포넌트**: JS만 작성

#### 미리보기
- **Preview 영역**에서 JS 코드 실행 결과 실시간 확인
- **에디터 레이어**에서 즉시 반영
- **Viewer 최종 결과**는 기존 이벤트 적용 후 반영

**워크플로우**:
```
코드 작성 → Apply 클릭 → Preview 반영 → 에디터 반영 → Viewer 반영
```

#### 코드 활용

**인스턴스 아이디로 HTML 요소 접근**:
```javascript
// 인스턴스 ID로 요소 선택
const element = document.getElementById('instance-id');
```

**CSS 스타일링**:
```css
#instance-id {
  background-color: #f0f0f0;
  padding: 20px;
}
```

**다국어 키값 변환**:
```javascript
// 다국어 키값은 __locale__.<key> 형식으로 자동 변환
const message = __locale__.greeting; // "안녕하세요" 또는 "Hello"
```

## 5. 통합 관리자 매뉴얼

### 개요
RENOBIT의 **통합 관리자 기능 및 설정 방법**을 다룹니다.

### 관리 모드 인터페이스

**구성 요소**:
- **ADMIN 로고**
- **사용자 정보 영역**
- **메뉴 영역**
- **헤더 영역**

**헤더 기능**:
- **다국어 변경** (한국어, 영어, 중국어, 일본어)
- **솔루션 이동** 버튼
- **메뉴 설정**
- **로그아웃**

### 사용자 관리

#### 코드 설정
- **부서 분류를 위한 코드 추가**
- 조직 구조 관리

**예시**:
```
부서 코드
├── DEV (개발팀)
├── DESIGN (디자인팀)
└── MARKETING (마케팅팀)
```

#### 사용자 추가/수정/삭제
**등록 정보**:
- **아이디**
- **비밀번호**
- **이름**
- **이메일**
- **부서**
- **권한 그룹**

#### 계정 활성/비활성화
- **잠긴 계정 해제** 기능
- 로그인 실패 횟수 초과 시 자동 잠금

#### 비밀번호 초기화
- 관리자가 **초기 비밀번호 설정**
- 사용자는 최초 로그인 시 변경 필요

#### 라벨 변경
- **이메일, 연락처 등 필드명 커스터마이징**
- 다국어별 라벨 설정

### 권한 그룹 관리
- **사용자별 권한 설정**
- **접근 페이지 제어**
- **관리자 그룹만 에디터 접근 가능**

**권한 예시**:
```
관리자 그룹
├── 에디터 접근: 허용
├── 관리 메뉴: 허용
└── 모든 페이지: 허용

일반 사용자 그룹
├── 에디터 접근: 거부
├── 뷰어: 허용
└── 특정 페이지만: 허용
```

### 라이선스 관리
**확인 정보**:
- **IP 주소**
- **만료 일자**
- **사용자/세션 라이선스 상태**

**관리 기능**:
- 라이선스 변경
- 갱신 요청
- 사용 현황 모니터링

### 보안 정책

#### 재인증
- **관리자 모드 접속 시 비밀번호 재확인**
- 보안 강화

#### 허용 IP
**지원 형식**:
1. **단일 IP**: `192.168.1.100`
2. **IP 범위**: `192.168.1.100-192.168.1.200`
3. **CIDR 표기**: `192.168.1.0/24`

**활용 사례**:
- 사무실 IP만 접근 허용
- VPN IP 대역 허용

#### 비밀번호 정책
**설정 항목**:
- **길이**: 최소 8자 이상
- **조합**: 영문, 숫자, 특수문자 조합
- **변경 주기**: 90일마다 변경 권장
- **이전 비밀번호 재사용 방지**

#### 휴면 계정
- **미사용 기간 후 자동 잠금**
- 보안 리스크 최소화
- 재활성화 절차

### 개인 환경 설정
- 사용자가 **개인 정보 수정**
- **메뉴 표시 여부 선택**
- 테마 설정 (추후 지원 예정)

### 통합 메뉴 관리
**기능**:
- **메뉴 순서 정렬**
- **메뉴 추가**
- **메뉴 삭제**
- **메뉴 동기화**

**관리 화면**:
```
관리자 페이지 메뉴
├── 사용자 관리
├── 권한 그룹 관리
├── 라이선스 관리
├── 보안 정책
└── 접속 로그 관리
```

### 접속 로그 관리
**기록 정보**:
- **사용자 계정 변경 이력**
- **접속 시간**
- **로그아웃 시간**
- **접속 사유** (필요시 기록)

**활용**:
- 보안 감사
- 사용자 활동 모니터링
- 문제 발생 시 추적

### 패키지 관리
**기능**:
- **컴포넌트, 플러그인 버전 관리**
- **호환성 확인**
- **ZIP 파일 업로드**로 설치/업데이트

**워크플로우**:
```
패키지 개발 → ZIP 압축 → 업로드 → 호환성 확인 → 설치 → 적용
```

**버전 관리**:
- 현재 버전 확인
- 업데이트 가능 버전 표시
- 롤백 기능

### 폰트 관리
- **기본 폰트 설정**
- **추가 폰트 관리**
- **지원 형식**: WOFF, WOFF2만 지원

**폰트 추가 절차**:
1. WOFF/WOFF2 파일 준비
2. 폰트 업로드
3. 폰트명 설정
4. 적용

**사용 예시**:
```css
@font-face {
  font-family: 'CustomFont';
  src: url('/resources/fonts/CustomFont.woff2') format('woff2');
}

body {
  font-family: 'CustomFont', sans-serif;
}
```
